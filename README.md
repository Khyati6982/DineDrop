# DineDrop 🍽️  
A Full‑Stack Food Ordering Platform  

DineDrop is a modern, recruiter‑ready full‑stack application designed for seamless restaurant browsing and online food ordering. Featuring a clean UI, secure payment integration, and robust backend automation, it delivers a smooth experience for both customers and admins.  

---

## ✨ Features  
🍴 **Restaurant Catalog** – Browse restaurants and their menus with real‑time availability.  
🛍️ **Cart & Checkout** – Add/remove dishes, manage cart, and complete purchases.  
💳 **Stripe Payment Integration** – Secure checkout with webhook‑based order updates.  
📦 **Order Management** – Orders auto‑update after successful payment.  
🛠️ **Admin Controls** – Admins can view complete order details and update payment status & order status.  
🎨 **Responsive UI** – Optimized for desktop, tablet, and mobile.  
🚀 **Deployment** – Backend deployed on Render, version control via GitHub.  

---

## 🛠️ Tech Stack Used  

### Frontend  
- **Thymeleaf** – Server‑side rendering for dynamic views.  
- **Bootstrap + Custom CSS** – Responsive styling and polished UI components.  

### Backend  
- **Spring Boot (Java 17)** – RESTful API routes and service layer.  
- **Spring Data JPA + Hibernate** – ORM for database persistence.  
- **Stripe Webhooks** – Automated payment confirmation and order updates.  

### Database  
- **MySQL** – Used locally and in Docker for development/testing.  
- **PostgreSQL (NeonDB)** – Used in production, integrated via Render/Vercel for cloud deployment.  

### Deployment & Tooling  
- **Render** – Backend hosting.  
- **Vercel + NeonDB** – Cloud database and environment integration.  
- **Docker** – Containerization for portability and local consistency.  
- **GitHub** – Version control and CI/CD integration.
